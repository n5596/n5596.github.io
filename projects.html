Projects

Automatic Essay Scoring (AES) with Bias Prediction

  • Developed an AES system using machine learning models (Bayes classifier, LSTMs, BiLSTMs) with a 97% agreement among scorers on the ASAP-AES dataset.
  • Processed data for feature extraction and trained models to predict the age and gender of the essay author to analyze the possibility of bias in essay scoring; indicated a possible gender bias in AES.

Each essay in our test set was assigned a score by the AES system. The age and gender prediction models were used to predict the demographic of the essay writer. Using the predicted demographic (age and gender) as well as the scores assigned, we analyzed the possibility of bias in automatic essay scoring. We wanted to know if certain words/phrases used by a particular demographic were scored higher than those used by others.

Technologies Used: Python, PyTorch, SkLearn

Book Management System

  • Conceptualized and executed a feature-rich application that offers personalized book recommendations to users based on their prior reading preferences, leveraging Flask-MySQL to generate HTML web pages.
  • Implemented a login functionality which allowed users to keep track of books read and the ratings they allotted to them.

A sign in functionality allows users to keep track of books they have read and assign a rating to them. The application suggests books matching user-defined criteria. If users have not specified any criteria, the application suggests a book which is similar to other books enjoyed by the user.

Technologies Used: Python, MySQL, Flask, HTML

Generational Training in Reinforcement Learning (RL)

  • Conceived a generational approach to train RL agents by exploiting the learning of the best performing agent from the previous generation; minimized runtimes by 50%.

Implemented a novel generational approach to train RL agents as well as a baseline approach which makes use of a shared buffer. The 2 approaches have been implemented for the Cartpole-v0 and Space Invaders-v0 gym environments of Open AI.

Technologies Used: Python, PyTorch
